# Medical-Store-Management
Medical Store Management System
A simple and modular C programming project designed for managing the inventory of a medical store.
This project allows users to add, search, update, delete, and display medicine records using clean
file handling, structures, and modular programming practices.
Project Overview
The Medical Store Management System focuses on inventory and stock management only (no billing),
keeping the implementation simple and within beginner-level scope.
Repository Structure:
/src - All .c files
/include - All .h files
/docs - ProjectReport.pdf
/assets - Screenshots
sample_input.txt
README.md
Features:
- Add new medicines
- Display all medicines
- Search medicine by ID
- Update quantity or price
- Delete a medicine
- Auto-save inventory to file
- Loads data from file at startup
Compilation:
gcc src/*.c -I include -o medical
Run:
./medical
Documentation:
ProjectReport.pdf inside /docs.
